🐤 Flappy Bird Remake

Java, JavaFX | Oct 2021 – Dec 2021

A desktop reimplementation of the classic Flappy Bird game using JavaFX, featuring smooth animations, difficulty scaling, and playful UI effects.

🔧 Key Features

  • Core gameplay mechanics: jumping physics and pipe generation
  • Score tracking and high score saving
  • Custom sound effects and increasing difficulty with progress
  • Responsive design using JavaFX layout controls and animation

👩‍💻 My Role

  • Designed the overall game architecture and screen navigation
  • Implemented animation loops and collision logic
  • Tuned user experience with fun audio cues and interface polish
