📚 Study with Jorge App

React Native, MongoDB | Feb 2024 – May 2024

Study with Jorge is a cross-platform mobile app that helps students form online study rooms, select seats, and track study habits with gamified features.

🔧 Key Features

  • Real-time seat selection in virtual rooms
  • Pomodoro-style study session tracking
  • User profiles with habit data visualization
  • Community features with chat and badges

👩‍💻 My Role

  • Built core frontend logic in React Native
  • Managed a small dev team and project milestones
